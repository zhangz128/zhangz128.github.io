Stock Data ETL Pipeline & Dashboard

  • Python
  • Airflow
  • Google Cloud

This project aimed to address the inefficiency of manual data retrieval and analysis in financial decision-making. I developed an automated system using Python to pull stock history, financial, and news data from the Yahoo Finance API. The system automates daily data extraction, allowing users to control tasks through an Airflow web server, transforms raw data into three structured tables, and loads them into a MySQL database. The entire system was deployed on Google Cloud, using Compute Engine for hosting. It also includes a stock history visualization dashboard and a newsletter page in Tableau for the latest updates.

R Package for Carbon Intensity Data

  • R
  • Data Visualization

The package helps users analyze and visualize carbon intensity across the United Kingdom. It uses real-time data and forecasts from the Carbon Intensity API to gather carbon emission statistics from different regions. Designed for educators, students, and environmental advocates, the package provides accessible tools for understanding and exploring carbon emission trends. This project is motivated by the need to address climate change by offering clear visualizations and simplifying data access, solving the problem of scattered information. Future enhancements could include integrating interactive learning modules, incorporating simulations for carbon reduction strategies, and expanding the scope to include global carbon data for broader educational impact.
